Welcome!

This is a simple start page you can publish with GitHub Pages.

Quick tips

  • Create more pages by adding new .html files (for example about.html).
  • Link pages using relative links: <a href="about.html">About</a>.
  • Put shared assets (CSS, images) in an assets/ folder and link them from your pages.

Publish

  1. Push these files to GitHub (commit and push to your main branch).
  2. Go to your repository on GitHub → Settings → Pages, and select the branch (usually main) and folder (root) to publish.
  3. Your site will be available at https://<username>.github.io/<repo>/ shortly after enabling Pages.
